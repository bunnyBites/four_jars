# four_jars

Four Jars is your personal finance sidekick, designed to make budgeting simple, intuitive, and maybe even a little fun. Say goodbye to complicated spreadsheets and hello to a beautifully streamlined approach to managing your money.

## Changelog

### Alpha Version 0.1
- **A Customizable Budget**: Users can set their own monthly income and allocation percentages.
- **Detailed Transaction Tracking**: Full support for categories and user-managed sub-categories.
- **Full Data Control**: Complete CRUD (Create, Read, Update, Delete) functionality for both transactions and sub-categories.
- **Persistent Storage**: All data is saved securely on the device using Hive, so nothing is lost when the app closes.
- **An Insightful Dashboard**: A dynamic pie chart and a clean grid layout provide an at-a-glance view of spending.
- **A Clean Architecture**: We separated the UI, logic, and data, making the app scalable and easy to maintain.
